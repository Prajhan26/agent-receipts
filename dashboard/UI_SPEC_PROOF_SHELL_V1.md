# ProofShell v1

This document locks the current `dashboard` landing page UI as a named reference version.

Use this name in future prompts when you want to preserve or return to this design:

`ProofShell v1`

## Intent

A dark, technical product-shell landing page for AgentReceipts with:

- a proof-first split hero
- a terminal-style quick start panel on the left
- a strong product value proposition on the right
- restrained green accents
- sharp bordered sections instead of soft marketing cards

## Defining Traits

### Hero

- Left panel: animated quick-start terminal
- Right panel: product headline, short explanation, CTA
- Layout: split product shell with clear border framing

### Visual Language

- Background: near-black
- Primary text: warm off-white
- Accent: neon green for verification/success
- Borders: thin, subtle, structured
- Surfaces: flat dark panels

### Typography

- Heading/display font: `Syne`
- Technical/meta font: `Space Mono`

### Product Feel

- more infrastructure/product than poster/branding
- more system panels than feature cards
- terminal/proof UI used as a trust signal

## Included Sections

1. top navigation with status pill
2. split hero with quick-start terminal
3. workflow section with 3 bordered steps
4. live receipts section
5. tamper demo section
6. footer
